Este proyecto implementa un sistema de sorteo aleatorio para un torneo de fútbol, desarrollado en Java siguiendo principios de Programación Orientada a Objetos (POO).
El usuario puede seleccionar una etapa del torneo (octavos, cuartos, semifinal o final), ingresar los equipos participantes y obtener los emparejamientos generados mediante un algoritmo recursivo.

Además, los resultados del sorteo se guardan en un archivo .txt dentro del proyecto para su consulta posterior.

Este trabajo forma parte de la entrega académica de la Unidad 3.
